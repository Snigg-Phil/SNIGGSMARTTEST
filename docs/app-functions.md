# App functions

You can find our app here:  
[Snigg Charging Manager App for Android](https://play.google.com/store/apps/details?id=com.cfos_emobility.charging_manager)  
[Snigg Charging Manager App for iOS](https://apps.apple.com/us/app/cfos-charging-manager/id1597082706#?platform=iphone)

The app has the following functions:

### registering the Snigg Snigg Smart Charger in the WLAN

The app finds the current WLAN in which your mobile phone is registered (permission is required for it to access the geolocation of your mobile phone). After you have entered the WLAN password, the app attempts to register the EVSE (via IP broadcasts) in this WLAN.  
Open the menu and click on "Connect to WLAN". Your PC must be connected to your router via WLAN. Enter the WLAN password in the "Password" field. The receiver device ID will be used later. Switch on all Snigg Snigg Smart Chargers that you want to connect to the WLAN.  
If you now click on "Send SSID and password", your EVSE password will be sent to all EVSEs that then connect to the WLAN (unless they are already connected to a WLAN).

**Note**: The WLAN password is encrypted using a general key. Theoretically, it is conceivable that someone knows this general key and can eavesdrop on your WLAN to determine the WLAN password. If you have security concerns about this, it is better to configure the Snigg Snigg Smart Controller manually by logging onto its access point. At a later time, you can optionally enter a "receiver device ID" so that the password is encrypted with this in a tap-proof manner. The disadvantage, however, is that you can then only register one Snigg Snigg Smart Controller per run in your WLAN.

### Find devices on the LAN

After your devices are logged into the network, the app can find them. Then you can assign a role (master or slave) to each device and have the EVSE and meter set up.

To do this, enter the admin password of the devices (the default is a blank password on delivery). Click on "Find devices". All devices registered in the home network will appear. You can now add devices to the system or switch to its web interface to configure it.

### Snigg Cloud

Using the Snigg Cloud, you can access your Snigg Snigg Smart Charger and the Snigg Charging Manager with the app (and also via a web browser) while on the move. Here are the [instructions for setting up cloud access](/en/cfos-charging-manager/documentation/cloud.htm).

Note: You can also use the remote maintenance feature built into the Snigg Charging Manager via the Snigg Cloud and administer devices that offer a suitable web interface while on the move.
